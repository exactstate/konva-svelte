# konva-svelte

WIP component library for KonvaJS (https://konvajs.org/)

## Usage

See [+page.svelte](https://github.com/exactstate/konva-svelte/blob/master/src/routes/%2Bpage.svelte) for example usage.

## Supported Components

- Arc
- Arrow
- Circle
- Ellipse
- Group
- Image
- Label
- Layer
- Line
- Path
- Rect
- RegularPolygon
- Ring
- Shape
- Sprite
- Star
- Text
- TextPath
- Transformer
- Wedge
